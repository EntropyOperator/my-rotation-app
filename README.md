# 画像暗記ローテーションアプリ

1日1枚、登録した画像を順番に表示するシンプルなWebアプリです。Notionなどの埋め込み（Embed）に最適化されています。

## セットアップ手順

### 1. 準備
1. このリポジトリをGitHubにアップロードします。
2. 忘れずに `images/` フォルダに必要な画像（`.png`, `.jpg` など）を入れてください。

### 2. コードの更新
`index.html` 内の `CONFIG` オブジェクトを編集して、追加した画像名をリストに登録します。

```javascript
const CONFIG = {
    baseDate: new Date('2024-01-01'), // 基準日（この日から何日経過したかで計算します）
    imageFolder: 'images/',
    images: [
        '001.png',
        '002.png',
        '003.png',
        // ここにファイル名を追加してください
    ]
};
```

### 3. 公開 (GitHub Pages)
1. GitHubリポジトリの **Settings** > **Pages** を開きます。
2. **Build and deployment** の **Branch** で `main` (または `master`) を選択して **Save** します。
3. 数分待つと公開用URLが発行されます。

### 4. Notionに埋め込む
1. Notionで `/embed` と入力します。
2. GitHub PagesのURLを貼り付けて **Embed link** をクリックします。

---

## 仕様
- **ローテーション**: `(今日 - 基準日) % 画像数` で計算されます。毎日1つずつ進みます。
- **レスポンシブ**: 画面サイズに合わせて画像がリサイズされます。
- **ダークモード**: OSの設定に合わせて背景色が自動で切り替わります。
